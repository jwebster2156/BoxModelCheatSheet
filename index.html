CSS Box Model Reference Sheet

Height and Width

Property Description
height How tall an element's box is
width How wide an element's box is

Example:

p {

height: 180px;

width: 200px;

}

Borders

Property Values Description Keywords and Units
border width The thickness of the border thin, medium, thick, or set in pixels
style The design of the border none, dotted, solid, and others
color The color of the border Over 140 built in color keywords; can also specify with RGB or hexadecimal notation
border-radius Affects the curvature of the corners by setting radius of corners Set in pixels or percent of width of box

Border example:

p {

border: 3px solid red;

}

Border-radius example:

p {

border-radius: 5px;

}

Padding

Property Description Example
padding Specifies the space between the contents of the box and its border on all four sides padding: 10px;
padding-top Specifies the padding on the top side padding-top: 10px;
padding-right Specifies the padding on the right side padding-right: 10px;
padding-bottom Specifies the padding on the bottom side padding-bottom: 10px;
padding-left Specifies the padding on the left side padding-left: 10px;

Padding Shorthand

# of Values Order of Values Example
4 Values top -- right -- bottom -- left padding: 6px 11px 4px 9px;
3 Values top -- right/left -- bottom padding: 5px 10px 20px;
2 Values top/botton -- right/left padding: 5px 10px;
